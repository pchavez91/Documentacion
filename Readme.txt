Significado de cada extencion 

| Lenguaje       | Extensión principal       | Descripción breve                                       |
| -------------- | ------------------------- | ------------------------------------------------------- |
| **C**          | `.c`, `.h`                | `.c` para código fuente, `.h` para headers.             |
| **C++**        | `.cpp`, `.hpp`, `.h`      | `.cpp` código, `.hpp/.h` headers.                       |
| **C#**         | `.cs`, `.csproj`, `.sln`  | `.cs` código, `.csproj` proyectos, `.sln` solución.     |
| **Java**       | `.java`, `.class`, `.jar` | `.java` fuente, `.class` compilado, `.jar` empaquetado. |
| **JavaScript** | `.js`, `.mjs`             | Código fuente JS; `.mjs` para módulos.                  |
| **TypeScript** | `.ts`, `.tsx`             | `.ts` para TS puro, `.tsx` para React.                  |
| **Python**     | `.py`, `.pyc`             | `.py` fuente, `.pyc` compilado.                         |
| **HTML**       | `.html`, `.htm`           | Archivos web estáticos.                                 |
| **CSS**        | `.css`, `.scss`, `.sass`  | Estilos (puros o preprocesadores).                      |
| **PHP**        | `.php`, `.phtml`          | Archivos de servidor con PHP.                           |
| **SQL**        | `.sql`                    | Scripts SQL.                                            |
| **Shell/Bash** | `.sh`, `.bash`            | Scripts de shell.                                       |
| **PowerShell** | `.ps1`                    | Script de PowerShell.                                   |
| **Go**         | `.go`                     | Archivos fuente de Golang.                              |
| **Rust**       | `.rs`                     | Archivos fuente en Rust.                                |
| **Kotlin**     | `.kt`, `.kts`             | Kotlin tradicional y Kotlin Script.                     |
| **Swift**      | `.swift`                  | Archivos de código en Swift.                            |
| **Ruby**       | `.rb`                     | Código fuente en Ruby.                                  |
| **Perl**       | `.pl`, `.pm`              | Código y módulos en Perl.                               |
| **Markdown**   | `.md`                     | Documentación estructurada.                             |
| **YAML**       | `.yaml`, `.yml`           | Configuraciones (por ejemplo, GitHub Actions, CI/CD).   |
| **JSON**       | `.json`                   | Datos o configuraciones.                                |
| **XML**        | `.xml`                    | Configuración, datos estructurados.                     |



**Tips Extra**

| Propósito               | Extensión    | Lenguaje o uso común        |
| ----------------------- | ------------ | --------------------------- |
| Configuración de VSCode | `.vscode/`   | Carpeta oculta con ajustes. |
| Archivos de entorno     | `.env`       | Variables de entorno.       |
| Docker                  | `Dockerfile` | Imagen y entorno Docker.    |
| Makefile                | `Makefile`   | Automatización de build.    |
| Git                     | `.gitignore` | Ignorar archivos/carpetas.  |
